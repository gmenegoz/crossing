# crossing

## Funzionalità implementate, da testare
- L'applicazione si apre all'indirizzo https://crossing-production.up.railway.app/
- Cliccando su "Random" viene creata una chimera con pezzi casuali
- Cliccando su "Manual" si apre una finestra per selezionare i pezzi
- Cliccando su EDIT si può modificare una chimera
- Cliccando su DELETE si può eliminare una chimera
- Cliccando su DOWNLOAD si può scaricare il file SVG da importare in Scratch
- Cliccando su "Import" si può caricare una cartella di file png (i file devono contenere il nome del layer nel loro nome)
- I nuovi pezzi caricati si possono usare per creare nuove chimere
- facendo doppio click sul nome di una chimera la si può rinominare
- Cliccando il pulsante di refresh della barra degli strumenti le chimere non si perdono
- Chiudendo e riaprendo la tab del browser le chimere non si perdono
- Collegandosi da browser/computer diversi si vedono chiemere diverse
- Dopo 10 minuti le chimere si perdono (ovviamente questo valore lo cambiamo poi)
